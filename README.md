# Samsung Music Player

Spotify's Android music player service — handles playback, playlists, and track recommendations using the legacy recommendation engine.

## Quick Start

```bash
npm install
npm run seed
npm start
```

## Recommendations

This service uses `music-player-recommendation-engine@3.x` for track recommendations.
See the [migration guide](docs/index.md) for upgrading to the new AI-powered engine.
